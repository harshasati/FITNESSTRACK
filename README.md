# FitnessTrack 💪

A full-stack MERN fitness tracking application that allows users to track workouts, view dashboards, and write fitness blogs.

---

## 🚀 Live Demo

Frontend: https://fitnesstrack-der5.vercel.app  
Backend API: https://fitnesstrack-kkok.onrender.com

---

## 🛠 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Vercel (Frontend Deployment)
- Render (Backend Deployment)

---

## ✨ Features

- User Authentication (JWT)
- Workout Tracking
- Dashboard Analytics
- Blog Creation
- REST API Integration
